# Variáveis Aleatórias

Valores dependem de eventos aleatórios
- lançamento de um dado (chance)
- tempo de vida útil de uma lâmpada incandescente

tipo:
Discretas
	Podem ser obtidos de uma contagem
	Ex: Nota de uma prova de matemática 

Contínuas
	São obtidos de uma medição
	Ex: Altura de um prédio residencial
